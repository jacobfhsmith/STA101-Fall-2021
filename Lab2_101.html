Visualizations

Here we will use the mtcars data set in the datasets package. There are a variety of ways to plot data in R. I provide an example of the simple plot command below, but for the rest of this lab, we will focus on using ggplot2.

(Note: When calling data, you can either use data = mtcars or type mtcars$mpg~mtcars$wt. This resource discusses common operators in R that may be helpful as a reference.)

plot(mpg ~ wt, data = mtcars) # formula notation

Now, let’s use ggplot2 to make a (in my opinion) more aesthetically pleasing version of this plot.

# Using ggplot:
library(ggplot2)
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point()

Basics of ggplot

The ggplot2 package has seven aspects, but we are going to focus mostly on the first three of the below for this lab:

  • Data
  • Aesthetics
  • Geometries
  • Facets
  • Statistics
  • Coordinates
  • Themes

Think about each aspect as a layer of your plot.

The first layer is your data, which makes sense intuitively, since you need to tell R which data set you are drawing from.

The second layer is aesthetics or mapping, which is how you want to plot the data, i.e., which variables you will use. It is specified inside aes().

Finally, you should decide which kind of plot you want for your data, such as a scatter plot, histogram, box plot, or bar plot.

(Note: a fourth aspect, geometries, are chosen via a series of geom functions, such as geom_point() for scatter plots.)

ggplot(data = mtcars, aes(x = hp, y = mpg, size = cyl)) + 
  geom_point(color = 'blue') + 
  ggtitle("Scatterplot")

Examples

Scatterplot

Scatter plots are best for showing a relationship between two numeric variables, as shown above.

Histogram

Histograms are used to display the distribution of numeric variables. Here, you will plot the variable of interest on the x-axis. The y-axis will show the frequency of the variable. You can adjust the plot with bin or binwidth.

mtcars$cyl = as.character(mtcars$cyl)
ggplot(data = mtcars, aes(x = mpg, fill = cyl)) + 
  geom_histogram(bins = 15) +
  ggtitle("Histogram")

Dot plots and box plots

Dot plots are useful when you want to show the distribution of either numeric or categorical variables.

ggplot(mpg, aes(x = class, y = displ)) + 
  geom_dotplot(binaxis='y', stackdir='center') +
  ggtitle("Dot plot")

Box plots are another way of showing distribution of numeric variables, with the 1st and 3rd quartiles represented by the edges of the box.

ggplot(mpg, aes(x = class, y = hwy)) + 
  geom_boxplot() + 
  ggtitle("Box plot")

Bar plot

A bar plot illustrates data in rectangular bars with length of the bar proportional to the value of the variable.

Below, we use dplyr to manipulate the data. Both ggplot2 and dplyr are tidyverse packages.

You will learn more about this package in next week’s lab (and in class on Monday the 6th), but here is a cheat sheet for how to use this package.

library(dplyr)
sleep_mean = sleep %>% 
  group_by(group) %>% 
  summarise(Mean_extra_sleep = mean(extra))

ggplot(data = sleep_mean, aes(x = group, y = Mean_extra_sleep)) +
  geom_bar(stat="identity") + 
  ggtitle("Mean Extra Sleep for Different Treatments") + 
  labs(x = "Treatment", y = "Extra Sleep")

Density plot

Density plots are a continuous version of a histogram. They convey the same information, the distribution of numeric variables. Instead of bins, density plots use a kernel density estimate (smooth curve) to represent the distribution. If you compare the density plot with the histogram, you can see they are showing the same thing.

ggplot(data = mtcars, aes(x = mpg, fill = cyl)) + geom_density() + ggtitle("Density plot")

As a reminder, if you don’t know something about a function or a package, use ? followed by the name of the function or package and R will give you documentation about it. Google is also a great resource for finding information about R, as is Stack Overflow.
